AngularJS Modules

We can have multiple ng-app directives in an HTML document but only one app is auto initialized and the rest needs to be manually initialized.

  • An angularjs module defines an application
  • A module is a container for different parts of our application.
  • A module is a container for the application controllers
  • A controller is always belongs to a module
  • We cannot declare a multiple modules in ng-app, this is a limitation of ng-app, it restricts we to only one module per page
  • main purpose: A module is a collection of directives, filters, services, controllers and configure information.
  • What is the purpose of angular module in AngularJS? NgModules consolidate components, directives, and pipes into cohesive blocks of functionality, each focused on a feature area, application business domain, workflow, or common collection of utilities. Modules can also add services to the application.
