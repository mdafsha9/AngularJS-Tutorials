AngularJS

What is AngularJS?

AngularJS is a JavaScript framework. It can be added to HTML with script tags.AngularJS extends HTML attributes with directives and it binds data to HTML with expression or ng-bind directive.
It runs on plain JavaScript and HTML, so we dont need any other dependencies to make it work. AngularJS is perfect for SPA.It is basically used to bind JavaSCript objects with HTML UI elements.

architecture of AngularJS

AngularJS is architecture on 3 components.They are:

  • the template(view), which is the HTML
  • the scope(model), which is the data available for the current view
  • the controller(controller), which is the JavaScript function that makes/changes/removes/controls the data.

How do you share data between controllers in AngularJS ?

We can share data by creating a service. Services are easiest, fastest and cleaner way to share data between controllers in AngularJS.
There are also ways to share data between controllers, they are
1.Using Events,
2.$parent, nextSibling, controllerAs, 3.Using the $rootScope

AngularJS Difgest cycle

AngularJS digest cycle is the process behind AngularJS data binding. In each digest cycle, Angular compares old and new versions of scope model values. The digest cycle is triggered automatically. we can also use $apply(), if we want totrigger the digest cycle manually.

differnce between AngularJS and JavaSCript expressions

  • Both can contain literals, operators and variables
  • AngularJS expressions can be written in HTML but JavaScript expressions are not.
  • AngularJS expressions do not support condiitonals, loops and exceptions but JavaScript expressions can do.
  • AngularJS expressions support filter but JavaScript expressions do not
