Custom directives

AngularJS provides built-in directives which offers functionality to our application.In AngularJS we can create our own custom directives also.
New directives are created by using .directive function. To invoke a directive, make an HTML element with the same tag name as a new directive.
By naming a custom directive we must use camelCase but when invoking is we must use "-" seperated name.

We can invoke a directive by using:

  • {{naming}}
