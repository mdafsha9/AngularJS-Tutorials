AngularJS Directives

  • AngularJS is used to extend HTML attributes with ng-directives and bind dat to HTML using expressions or ng-bind directive.
  • AngularJS has set of built-in directives which offers functionality to our application.
  • AnglarJS lets we create our own directives
  • ng-app directive is a root element of an Angularjs application and it will auto-bootstrap (automatically initializes) the application, when a web page is loaded
  • ng-model directive binds the values of HTML controls (textarea, input, select) the application data
    • ng-model also
    • provide type validation for application data(email, required, number)
    • provide status for application data (invalid, dirty, touched, error)
    • provide css classes for HTML elements
    • provide HTML elements to HTML forms
  • ng-bind directive binds aplication data to HTML view
  • ng-init defines the angularjs application variables
  • ng-controller defines angularjs controller
  • ng-repeat directive repeats an HTML element. Actually ng-repeat directive clones HTML element once for each item in a collection

Creating our own directive

  • Create a new directive by using .directive function
  • To invoke a new directive, make an HTML element with the same tag name as new directive
  • When naming a new directive, we must use camelCase when invoking it, we must use - seperated name
  • WE can invoke a directive by using
    • element name
    • attribute
    • class
    • comments
