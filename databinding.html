AngularJS Data Binding

Data binding in angularJS is the synchronization between the model and the view.
In angularjs application usually have a data model. The data model is a collection of data available for the application.

One-way-data binding
Changes in the model are reflected in the view but chnages in the view are not reflected in the model. The binding is one-way from the model to view.This is achieved by making use of ng-bind directive
Two-way-data binding
Changes in the model are reflected in the view as well as changes in the view are reflected in the model.This is acheived by making use of ng-model directive

AngularJS Controllers

  • Angularjs controllers control the data of AngularJS application
  • Angularjs controllers are regular Javascript objects
  • Angularjs application are controlled by controllers
  • ng-controller define angularjs controllers
  • Controller is a Javascript object it is created by standard javascript object constructor
  • AngularJS will invoke the controller with $scope object
