AngularJS

What is AngularJS?

AngularJS is a JavaScript framework. It can be added to HTML with script tags.AngularJS extends HTML attributes with directives and it binds data to HTML with expression or ng-bind directive.
It runs on plain JavaScript and HTML, so we dont need any other dependencies to make it work. AngularJS is perfect for SPA.It is basically used to bind JavaSCript objects with HTML UI elements.

How do you share data between controllers in AngularJS ?

We can share data by creating a service. Services are easiest, fastest and cleaner way to share data between controllers in AngularJS.
There are also ways to share data between controllers, they are
1.Using Events,
2.$parent, nextSibling, controllerAs, 3.Using the $rootScope
